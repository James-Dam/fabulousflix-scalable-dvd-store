# Course Milestones

This project was completed across multiple course milestones.

## Project 1 — Core Movie Browsing

Implemented the initial Java/Tomcat/MySQL web application with movie, star, and movie-list pages.

Demo: [[YouTube link](https://www.youtube.com/watch?v=vZTTsqC6jso&t=19s&ab_channel=ColinHarrison)]

## Project 2 — Search, Browse, Cart, and Sessions

Implemented search, browsing, login/session handling, pagination, filtering, shopping cart functionality, and checkout flow.

Demo: [[YouTube link](https://www.youtube.com/watch?v=e4ULRRYqZZ4&t=2s)]

## Project 3 — XML Parsing, Security, and Employee Dashboard

Added XML data ingestion, HTTPS, reCAPTCHA, stored procedures, and an employee dashboard.

Demo: [[Youtube link](https://www.youtube.com/watch?v=qYTMTcwylLU&ab_channel=ColinHarrison)]

## Project 4 — Scaling and Replication

Added JDBC connection pooling, MySQL master-slave replication, and load-balanced deployment.

Demo: [[Youtube link](https://youtu.be/ftSkSJjwR7s)]

## Project 5 — Docker/Kubernetes Deployment

Containerized the application and migrated it toward a multi-service architecture.

Demo: [[Youtube link](https://youtu.be/ftSkSJjwR7s)]
